## Setup

Install dependencies.

```sh
npm install
```

## Usage

### Format

Format code using [Prettier](https://prettier.io/).

```sh
npm run format
```

### Linting

Lint code using [ESLint](https://eslint.org/).

```sh
npm run lint
```

### Testing

Run tests using [Jest](https://jestjs.io/) testing framework.

```sh
npm test
```
